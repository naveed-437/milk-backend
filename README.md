ASN Dairy Hub
===============

Simple dairy distribution management app — Node/Express backend with Supabase and React + Vite frontend.

What it is
---------

- Manage milk products, customers, deliveries and subscriptions.
- Built to demonstrate a full-stack CRUD dashboard with uploads, auth, and production static serving.

Architecture
------------

- Backend: Node.js + Express, Supabase JS client for data.
- Frontend: React + Vite, single-page app with protected routes and a small dashboard.

Quick Start (development)
-------------------------

1. Install dependencies (root runs backend scripts that call frontend build/dev):

```powershell
cd c:\milk-backend
npm install
cd frontend
npm install
```

2. Run backend in development (uses `server.js`):

```powershell
cd c:\milk-backend
npm run dev
```

3. Run frontend dev server (optional):

```powershell
cd c:\milk-backend\frontend
npm run dev
```

Production build & serve
------------------------

1. Build frontend and start backend which serves `frontend/dist`:

```powershell
cd c:\milk-backend
npm run build
npm start
```

2. Open `http://127.0.0.1:5000` in your browser.

Features
--------

- Products: CRUD, image uploads
- Customers: CRUD
- Deliveries: Create scheduled deliveries using customers/products
- Subscriptions: Manage recurring subscriptions

In-app Help
-----------

Open the `Help` page from the top navigation in the app for an interactive overview and sample flows.

Demo data, docs, and deployment
-------------------------------

- Seed demo data with `npm run seed`.
- Review API usage in [API_DOCS.md](API_DOCS.md).
- Build a container image with `docker build -t milk-backend .`.
- Run the smoke test with `npm test`.

Contributing
------------

PRs are welcome. For changes to the frontend, edit files under `frontend/src` and run `npm run build`.
